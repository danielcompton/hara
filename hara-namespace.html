namespace

utilities for manipulation of namespaces

1    Introduction

hara.namespace contain namespace functions and utilities

1.1    Installation

Add to project.clj dependencies:

[im.chit/hara.namespace "2.3.0"]

Individual namespaces can be added seperately:

[im.chit/hara.namespace.eval "2.3.0"]
[im.chit/hara.namespace.import "2.3.0"]
[im.chit/hara.namespace.resolve "2.3.0"]

2    API - eval

hara.namespace.eval

API

eval-ns  eval-temp-ns  with-ns  with-temp-ns  

eval-ns ▴

"Evaluates a list of forms in an existing namespace" (eval-ns 'hara.common.checks '[(long? 1)]) => true

eval-temp-ns ▴

"Evaluates a list of forms in a temporary namespace" (eval-temp-ns '[(def inc1 inc) (defn inc2 [x] (+ 1 x)) (-> 1 inc1 inc2)]) => 3 "All created vars will be destroyed after evaluation." (resolve 'inc1) => nil

with-ns ▴

"Evaluates `body` forms in an existing namespace given by `ns`." (require '[hara.common.checks]) (with-ns 'hara.common.checks (long? 1)) => true

with-temp-ns ▴

"Evaluates `body` forms in a temporary namespace." (with-temp-ns (def inc1 inc) (defn inc2 [x] (+ 1 x)) (-> 1 inc1 inc2)) => 3 "All created vars will be destroyed after evaluation." (resolve 'inc1) => nil

3    API - import

hara.namespace.import

API

import  import-namespace  import-var  

import ▴

"Imports all or a selection of vars from one namespace to the current one." (import hara.common.checks [bytes? long?]) => nil (eval '(long? 1)) => true (eval '(bytes? 1)) => false (import hara.common.checks :all) => nil (eval '(bigint? 1)) => false

import-namespace ▴

"Imports all or a selection of vars from one namespace to the current one." (import-namespace 'hara.common.checks '[bytes? long?]) (eval '(long? 1)) => true (eval '(bytes? 1)) => false

import-var ▴

"Imports a single var from one namespace to the current one." (import-var 'ifl #'clojure.core/if-let) => anything ; #'hara.namespace.import-test/ifl (eval '(ifl [a 1] (inc a))) => 2

4    API - resolve

hara.namespace.import

API

ns-vars  resolve-ns  

ns-vars ▴

"lists the vars in a particular namespace" (ns-vars 'hara.namespace.resolve) => '[ns-vars resolve-ns]

resolve-ns ▴

"resolves the namespace or else returns nil if it does not exist" (resolve-ns 'clojure.core) => 'clojure.core (resolve-ns 'clojure.core/some) => 'clojure.core (resolve-ns 'clojure.hello) => nil
