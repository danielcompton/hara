1    Introduction

hara provides a set of functions, best practises and code abstractions. It serves a clojure.contrib like purpose but attempts to uphold the higher principles of programming.

  • synergistic design
  • maximal code reuse
  • minimal code repetition and wastage
  • functional orthogonality
  • functional modularity
  • functional extensibility
  • self-documentated code

Please See finding a middle ground for motivations and reasoning.

3.7    common.watch

e.3.37  -  add

"Adds a watch function through the IWatch protocol"

(let [subject (atom nil)
      observer  (atom nil)]
  (watch/add subject :follow
             (fn [_ _ _ n]
               (reset! observer n)))
  (reset! subject 1)
  @observer => 1

  "Alternatively, options can be given to either transform the current
  as well as to only execute the callback if there is a difference."

  (let [subject  (atom {:a 1 :b 2})
        observer (atom nil)]
  (watch/add subject :clone
             (fn [_ _ p n] (reset! observer n))
             {:select :b
              :diff true})

  (swap! subject assoc :a 0) ;; change in :a does not
  @observer => nil           ;; affect watch

  (swap! subject assoc :b 1) ;; change in :b does
  @observer => 1))

e.3.38  -  list

"Lists watch functions through the IWatch protocol"

(let [subject   (atom nil)
      observer  (atom nil)]
  (watch/add subject :a (fn [_ _ _ n]))
  (watch/add subject :b (fn [_ _ _ n]))
  (watch/list subject) => (contains {:a fn? :b fn?}))

e.3.39  -  remove

"Removes watch function through the IWatch protocol"

(let [subject   (atom nil)
      observer  (atom nil)]
  (watch/add subject :a (fn [_ _ _ n]))
  (watch/add subject :b (fn [_ _ _ n]))
  (watch/remove subject :b)
  (watch/list subject)) => (contains {:a fn?})

e.3.40  -  clear

"Clears all watches form the object"

(let [subject   (atom nil)
      observer  (atom nil)]
  (watch/add subject :a (fn [_ _ _ n]))
  (watch/add subject :b (fn [_ _ _ n]))
  (watch/clear subject)
  (watch/list subject)) => {}

e.3.41  -  set

"Sets a watch in the form of a map"

(let [obj   (atom nil)]
  (watch/set obj {:a (fn [_ _ _ n])
                  :b (fn [_ _ _ n])})
  (watch/list obj) => (contains {:a fn? :b fn?}))

e.3.42  -  copy

"Copies watches from one object to another"

(let [obj-a   (atom nil)
      obj-b   (atom nil)]
  (watch/set obj-a {:a (fn [_ _ _ n])
                    :b (fn [_ _ _ n])})
  (watch/copy obj-b obj-a)
  (watch/list obj-b) => (contains {:a fn? :b fn?}))

"disassociates keys from a nested map. Setting `keep` to `true` will

"returns a map of all key/value pairs that differ from a second map"

(unique {:a 1} {:a 2})
=> {:a 1}

(unique {:a 1 :b 2} {:b 2})
=> {:a 1}

(unique {:b 2} {:b 2 :a 1})
=> nil

e.4.3  -  assoc-if

"assoc key/value pairs to the map only on non-nil values"

(assoc-if {} :a 1)

"assoc-in a nested key/value pair to a map only on non-nil values"

(assoc-in-if {} [:a :b] nil)

(merge-if {:a nil :b 1})

(merge-if {:a 1} {:b nil} {:c 2})
