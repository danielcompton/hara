Installation
 
In your project.clj add the following to include the entire hara library:
 
[im.chit/hara "2.2.15"]
 
Include selected functionality by including the specific package. For example, if the hara.io.watch namespace is required, use the following instead:
 
[im.chit/hara.io.watch "2.2.15"]
 
Usage
 
Depending on the functionality that you wish to use, import the require namespace and start coding:
 
(use 'hara.io.watch) (-> (watcher ["."] println {:types #{:create :modify} :recursive true :filter [".clj"] :async true}) (start-watcher))
 
More information about packages can be found in the api section and the guides.
