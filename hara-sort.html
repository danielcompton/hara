sort

micellaneous sorting functions

1    Introduction

hara.sort contain functions and utilities for strings.

1.1    Installation

Add to project.clj dependencies:

[im.chit/hara.sort "2.2.13"]

Individual namespaces can be added seperately:

[im.chit/hara.sort.topological "2.2.13"]

2    API - topological

hara.sort.topological

API

top-nodes  topological-sort  

top-nodes ▴

"nodes that have no other nodes that are dependent on them" (top-nodes {:a #{} :b #{:a}}) => #{:b}

topological-sort ▴

"sorts a directed graph into its dependency order" (topological-sort {:a #{:b :c}, :b #{:d :e}, :c #{:e :f}, :d #{}, :e #{:f}, :f nil}) => [:f :d :e :b :c :a] (topological-sort {:a #{:b}, :b #{:a}}) => (throws Exception "Graph Contains Circular Dependency: {:b #{:a}, :a #{:b}}")
